# Hugo sourcecode

### 1. Install Hugo

```sh
    sudo apt-get install hugo
```

- To verify your new install:


```sh
    hugo version
```

### 2. Create a New Site

```sh
    hugo new site quickstart
```

### 3. Start hugo

```sh
    hugo server -D
```

### 4. New post

```sh
    hugo new posts/my-first-post.md
```

### 5. Build static pages 

```sh
    hugo -D
```
